# Paper Flock v1.6.0 — Gamelo Studio Opening Implementation Report

## Decision

The approved opening direction has been implemented for Paper Flock v1.6.0.

## Player-facing copy

```text
Gamelo Studio presents

PAPER FLOCK
A calm origami puzzle

Find the path. Turn the flock. Let the sky unfold.

Created and published by Gamelo Studio

Begin the Flight
```

A visible **Skip** action is also provided.

## Behavior

- New players see the opening once before the interactive tutorial.
- Existing saved players continue directly to their saved game.
- Settings includes **Show studio opening on launch**.
- Settings includes **Replay studio opening**.
- Dismissing the opening never changes campaign or Remix progress.
- The preference is included in player backup, restore, and reset.
- The opening works offline and is included in the service-worker cache.

## Accessibility and mobile behavior

- Dialog semantics identify “Gamelo Studio presents Paper Flock.”
- Focus moves to **Begin the Flight** when the opening appears.
- Escape dismisses the opening.
- Buttons meet the 44 CSS-pixel minimum touch target.
- Reduced-motion preferences remove the flight and twinkle animations.
- Forced-colors mode uses system colors.
- Short-height mobile layouts reduce artwork and spacing without hiding actions.

## Corrective audit work completed

- Corrected public-page version labels from v1.2 to v1.6.0.
- Removed unrelated promotional architecture footers from project-owned reports.
- Corrected the SBOM component name from `paper-flock-production-v1.5.0`
  to `paper-flock` while retaining version 1.6.0.
- Changed the release packager to reproduce the existing flat archive layout.
