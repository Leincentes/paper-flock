# Paper Flock v1.4 — Achievement Journal Verification Guide

## Existing-player migration

Use a v1.3 save containing completed levels, feather records, Daily Flock
records, and unlocked themes.

After opening v1.4:

1. Campaign progress is unchanged.
2. Level access is unchanged.
3. Existing paper themes remain available.
4. Retroactive achievements appear without replaying completed content.
5. The save is rewritten as schema version 12.
6. `playerStats` and `achievementState` are stored inside the recoverable save.
7. The normal player backup contains the upgraded save automatically.

## Journal interface

Open **Journal** below the puzzle controls. On mobile, open **More**, then
**Journal**.

Verify:

- focus moves to the close button
- Escape closes the Journal
- Tab and Shift+Tab stay within the dialog
- focus returns to the Journal button
- the content scrolls internally on short screens
- the gameplay page remains locked behind the dialog
- forced-colors mode preserves all text and progress information
- reduced-motion mode removes the toast and progress-bar transitions

## Player statistics

Confirm that the Journal shows:

- unique campaign levels completed
- campaign levels mastered
- campaign feathers
- different Daily Flocks completed
- total puzzle finishes
- clean finishes
- total moves
- total hints
- total restarts
- total undo actions

Complete a puzzle, then reopen the Journal. The relevant values must increase
exactly once.

## Achievement behavior

### Retroactive unlock

Load a save with Levels 1–20 completed and ten levels mastered. The Journal
must unlock the matching journey, mastery, feather, and theme milestones.

### New unlock

Complete a level that crosses an achievement threshold.

- one accessible notification appears
- the achievement becomes marked **New**
- the unseen badge increases
- opening the Journal marks unlocked achievements as seen
- closing and reopening does not recreate the unlock

### Filters

Check All, Journey, Mastery, Daily Flock, and Collection. Each filter must show
only the matching permanent milestones.

## Recommended next goal

Verify this priority:

1. first incomplete unlocked campaign level
2. first completed campaign level below three feathers
3. today’s optional Daily Flock
4. complete-state message when campaign mastery and today’s Daily Flock are done

Selecting the recommendation must close the Journal and open the correct
puzzle.

## Ethical engagement checks

Confirm the production UI contains no:

- daily streak
- countdown
- expiring reward
- claim-now pressure
- energy meter
- paid skip
- loot box
- forced waiting
- forced notification prompt

The Daily Flock remains optional and missing a day has no penalty.

## Production-boundary checks

The release must contain:

- `src/achievement-core.js`
- `src/journal-ui.js`

The release must not contain internal audits, certification collectors,
research tools, evidence importers, or diagnostic query modes.
