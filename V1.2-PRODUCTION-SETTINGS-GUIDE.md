# Paper Flock v1.2 — Production Settings Guide

## Opening Settings

On desktop, select **Settings** below the game controls.

On mobile, select **More**, then **Settings**. The puzzle page remains locked
to the visible viewport while the Settings panel scrolls internally.

## Game settings

### Sound effects

Controls synthesized bird, fold, hint, blocked-move, undo, and completion
sounds. The preference is saved with player progress.

### Haptic feedback

Controls short device vibrations for supported mobile browsers and installed
apps. Disabling haptics does not affect sound or animation.

### Visual effects

- **Automatic:** adapts to reduced-motion preferences and device capability.
- **Full:** enables the complete visual treatment.
- **Lite:** reduces expensive decorative effects.
- **Minimal:** keeps essential state feedback with minimal animation.

### Paper theme

Lists every theme and disables themes that are not unlocked. Selecting an
available theme applies it immediately and saves the preference.

### Replay how to play

Closes Settings and opens the interactive tutorial without resetting campaign
progress.

## Accessibility settings

### Text size

Standard, Large, and Extra large settings apply across gameplay and dialogs.

### Contrast

Automatic follows the operating-system preference. Always high enables the
game's stronger contrast treatment.

### Motion

Automatic follows the operating-system preference. Reduced limits decorative
and repeated movement.

## App and data

### Status

Displays online/offline state and whether the game is running as an installed
app or in the browser.

### Install app

Appears only when the browser exposes an install prompt and Paper Flock is not
already installed.

### Install update

Appears only when a new service worker is waiting. Selecting it activates the
update and reloads the game.

### Export progress backup

Downloads a JSON file containing only:

- current and backup save envelopes
- accessibility preferences
- tutorial completion
- orientation-dismissal preference

It excludes internal QA, certification, release-evidence, and diagnostic data.

### Restore progress backup

Validates the product, schema, file size, and allowed storage keys before
asking the player to confirm replacement of current player data.

### Reset all player data

Removes player progress, tutorial completion, and preferences. It does not
delete unrelated browser storage.

## About

Shows Paper Flock v1.2, Gamelo Studio, the support email, and links to all
player-facing legal and release information.

## Keyboard and screen-reader behavior

- Focus moves into Settings when it opens.
- Tab and Shift+Tab remain inside the panel.
- Escape closes Settings.
- Focus returns to the control that opened it.
- Status changes use a polite live region.
- Controls use native checkboxes, selects, buttons, and links.

## Production verification

Confirm on desktop, Android, and iPhone:

1. Every preference persists after reload.
2. Haptics can be disabled independently of sound.
3. Locked themes cannot be selected.
4. Accessibility changes apply immediately.
5. Backups export, validate, restore, and preserve progress.
6. Reset requires confirmation.
7. Install and update buttons appear only when applicable.
8. The mobile page does not scroll behind Settings.
9. No prototype, testing, certification, or evidence controls appear.
