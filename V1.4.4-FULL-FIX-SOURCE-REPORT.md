# Paper Flock v1.4.4 — Full Fix Source Report

This source package consolidates all release fixes identified during GitHub qualification.

## Included fixes

1. Public npm registry lockfile with `speedline-core@1.4.3`.
2. Android WebView debugging detection without generated `BuildConfig`.
3. Android API 36 predictive-back migration using `OnBackPressedDispatcher`.
4. Activity Result API migration for backup import and export.
5. AndroidX edge-to-edge and window-insets migration.
6. Removal of deprecated WebSQL and legacy system-bar APIs.
7. Browser startup and modal-focus race-condition fixes.
8. API-qualified `android:windowLightNavigationBar` theme resource:
   - removed from the unqualified API 24 base resources
   - added under `values-v27`
   - repeated under `values-v31` alongside splash resources

## Expected GitHub checks

- `npm ci --no-audit --no-fund`
- Node/unit tests
- browser-quality with zero flaky tests
- `gradle -p android clean lintDebug bundleDebug --warning-mode all`
- signed release AAB after signing secrets are configured

## Remaining external verification

The Android SDK is unavailable in the artifact-generation environment, so the final
`lintDebug`, `bundleDebug`, and signed release bundle must be confirmed by GitHub Actions.
