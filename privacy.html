Local-first data use

Privacy notice

Paper Flock public beta · Build v0.16

Paper Flock does not automatically upload gameplay, feedback, diagnostics, or personal information.

Information stored on your device

The game may store progress, settings, active puzzle state, local playtest sessions, installation checks, feedback drafts, and local client-error records in browser storage. This supports offline play, exact resume, testing, and deliberate export.

Manual exports

Files leave your device only when you deliberately download and send a backup, diagnostic file, test result, certification report, or feedback report. Review free-text answers before sharing because you might enter personal information accidentally.

Accounts, advertising, and payments

This beta has no account, advertising SDK, behavioral analytics, purchase system, location collection, or advertising identifier.

Hosting

The static hosting provider may process ordinary technical request information under its own policies when it serves the game files. Paper Flock does not receive that information through an application analytics endpoint.

Deletion

Use Prototype testing tools → Reset local data, remove the site's browser data, or uninstall the Home Screen web app. Create a backup first when you want to keep progress.

Children

The public beta is not designed to collect information from children. Do not run formal tests with minors without appropriate guardian consent and policy review.

Changes

Effective 19 June 2026. Material changes will appear in the release notes and this page.
