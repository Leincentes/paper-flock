# Paper Flock v1.6.0 tester instructions

Thank you for testing Paper Flock. Please use the game naturally before reading
the detailed tasks.

1. Install from the private Google Play testing link.
2. Observe the Gamelo Studio opening, then use **Begin the Flight** or **Skip**.
3. Complete the tutorial without outside help.
4. Play at least five levels.
5. Confirm sound is on during the first session.
6. Close and reopen the game.
7. Turn on airplane mode and reopen it again.
8. Export a backup from Settings → Data, then restore it.
9. Open Settings → Data → Export tester report.
10. Continue toward Level 11 during the test period.
11. Install at least one closed-test update without clearing app data.
12. Open Settings → Game → **Replay studio opening**.
13. Enable **Show studio opening on launch**, relaunch once, then disable it.

Report any startup recovery screen, lost progress, blocked control, distorted
layout, missing sound, crash, Android “not responding” event, or confusing
instruction.

## When a problem occurs

1. Reproduce the problem once when safe.
2. Open **Settings → Data → Export tester report** immediately afterward.
4. Send the report only through the private channel provided by the test
   organizer.
5. Do not post the report publicly.
6. If the app cannot start, use the startup-recovery screen to export the
   tester report or raw recovery file.
7. Use **Start without changing progress** only as a temporary recovery option.
   It should not replace the normal saved game.

The app never uploads diagnostics automatically. You decide whether to export
or share a report.

Do not share the private opt-in link publicly.
