# Google Play submission checklist

## Account

- [ ] Create or verify the owner-controlled Play Console account
- [ ] Pay the one-time registration fee
- [ ] Complete personal or organization identity verification
- [ ] Verify developer email and phone
- [ ] Decide whether the public developer name is `Gamelo Studio`
- [ ] Confirm permanent package name: `com.gamelostudio.paperflock`

## Build

- [ ] Generate and securely back up the upload keystore
- [ ] Add GitHub Actions signing secrets
- [ ] Run the Android bundle workflow
- [ ] Download `paper-flock-v1.4.4.aab`
- [ ] Inspect the bundle in Play Console App bundle explorer
- [ ] Enroll in Play App Signing
- [ ] Confirm version code `10404`, version name `1.4.4`
- [ ] Confirm target SDK 36
- [ ] Confirm only VIBRATE permission is declared

## Store listing

- [ ] Upload 512×512 icon
- [ ] Upload 1024×500 feature graphic
- [ ] Upload at least three 1080×1920 phone screenshots
- [ ] Paste title, short description, and full description
- [ ] Add screenshot and feature-graphic alt text
- [ ] Add support email and website
- [ ] Publish and verify privacy-policy URL
- [ ] Optionally upload the real-gameplay YouTube preview video

## App content

- [ ] Complete Data safety form
- [ ] Declare no ads
- [ ] Complete IARC content-rating questionnaire
- [ ] Complete target-audience declaration
- [ ] Confirm no restricted permissions
- [ ] Confirm all app features are accessible without login
- [ ] Add reviewer instructions

## Testing and release

- [ ] Upload to internal testing first
- [ ] Install on at least one physical Android phone
- [ ] Complete backup, offline, update, and Level 11 lesson checks
- [ ] Start closed testing
- [ ] For a qualifying personal account, retain at least 12 opted-in testers
      continuously for 14 days
- [ ] Review pre-launch report
- [ ] Fix all critical and high-confidence issues
- [ ] Apply for production access when eligible
- [ ] Use staged rollout rather than 100% immediate release
