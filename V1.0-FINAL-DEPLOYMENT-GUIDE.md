## Final deployment

1. Push this package to `main`.
2. Wait for all unit, dependency, browser, Lighthouse, CodeQL, deployment,
   SBOM, and provenance jobs to pass.
3. Verify the live game at https://leincentes.github.io/paper-flock/.
4. Download `paper-flock-v1.0-quality-evidence`.
5. Import it under Prototype testing tools → Production release evidence.
6. Review all previously imported mobile, accessibility, and player evidence.
7. Type `RELEASE PAPER FLOCK 1.0`.
8. Archive the approval JSON, release ZIP, checksum, SBOM, and workflow evidence.

A v1.0 approval record is not pre-generated because the new release has a new
version, workflow run, and SHA-256 digest.
